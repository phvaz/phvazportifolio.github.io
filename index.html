Projeto Google Search

O projeto "Google Search" consiste em criar uma interface de usuário para a Pesquisa do Google utilizando HTML, CSS e formulários. O objetivo é replicar a interface de pesquisa do Google com uma caixa de pesquisa e um botão "Pesquisa Google". Os usuários podem digitar uma consulta de pesquisa, clicar no botão e serem redirecionados para a página de resultados de pesquisa do Google, com a consulta como um parâmetro GET. O projeto possui páginas para pesquisa de imagens e pesquisa avançada do google, com campos de entrada específicos para diferentes opções de pesquisa, além disso, há o botão "Estou com sorte" que redireciona o usuário para o primeiro resultado da pesquisa. O projeto visa simular as principais funcionalidades e o design visual da página de pesquisa do Google. Clique na imagem para ser redirecionado para um link no youtube demonstrando a funcionalidade do projeto ou no botão abaixo para ver seu código.

Filtro

Este projeto em C envolve a implementação de um programa que aplica filtros a imagens BMP. Os filtros disponíveis são conversão para escala de cinza, sépia e reflexão horizontal, além do efeito de desfoque. O programa lê arquivos BMP, processa as imagens e salva os resultados como novos arquivos BMP filtrados.

Eleição

Este projeto é uma simulação de uma eleição com segundo turno implementada em C. Ele permite aos usuários executarem um sistema de votação de escolha ranqueada, onde os eleitores podem classificar os candidatos por ordem de preferência. O processo de segundo turno é conduzido até que um candidato receba a maioria dos votos (mais de 50%). O programa possui funções para registrar os votos, calcular os totais de votos, determinar o vencedor e eliminar candidatos com poucos votos.

Biblioteca

O projeto é uma aplicação de gerenciamento de biblioteca desenvolvida em Python com interface gráfica (GUI) usando tkinter. Permite adicionar, buscar e exibir a disponibilidade de livros. As classes Book e Library são usadas para organizar as informações dos livros e as operações da biblioteca. Também inclui testes unitários para garantir o correto funcionamento das classes. A GUI intuitiva torna a interação com o sistema amigável e útil para gerenciar acervos bibliotecários.

Herança Sanguínea

Este projeto é uma simulação de herança de tipos sanguíneos em uma família, implementado em C. O programa cria uma árvore genealógica com gerações de indivíduos e atribui alelos aleatórios para representar os tipos sanguíneos. Os alelos são herdados dos pais, e a simulação mostra como os tipos sanguíneos podem ser transmitidos ao longo das gerações. O código utiliza uma abordagem recursiva para construir a árvore genealógica e é destinado para fins educacionais sobre genética e algoritmos de simulação.

Readability

Este projeto consiste na implementação de um programa em C chamado "readability", que determina o nível de leitura de um texto usando o índice de Coleman-Liau. O programa solicita ao usuário um texto e calcula o número médio de letras, palavras e frases no texto. Com base nessas informações, o programa aplica a fórmula do índice de Coleman-Liau para determinar o nível de leitura do texto, indicando qual série escolar é necessária para compreendê-lo.
